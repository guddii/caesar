# Caesar Encryption
A C++ project for caesar en- and decryption

[![Build Status](https://travis-ci.org/guddii/cpp-caesar.svg?branch=master)](https://travis-ci.org/guddii/cpp-caesar)

## Requirements

- Python v2.3 or newer
- CMake v2.6.4 or newer
- Clang v3.4 or newer

## Build steps
```
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
$ ctest -VV
```
